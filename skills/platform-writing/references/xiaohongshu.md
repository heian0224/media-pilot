# 小红书 (Xiaohongshu / RED) Writing Guide

Casual, visual, recommendation-driven ("种草"). The reader scrolls fast — you have the title and first image to earn the tap, then 300-800 words to deliver value and warmth.

## Voice & Tone
- **轻松/种草感** — like a friend recommending something, not a reviewer
- **第一人称亲测** — "我用了/我试了/我踩坑了" builds trust
- **具体到数字** — "3 天""涨了 500 粉""省了 2 小时" (小红书读者吃这套)
- **emoji 多用** — ✨🔥💡📝✅❤️ 自然融入，但别堆砌成乱码
- **口语化** — "姐妹们""绝绝子""真的会谢""谁懂啊"

## Length & Structure
- **300-800字** — 超过 800 多数人不读完
- **标题**: 20字内，必须吸睛。最高效的模板：
  - 数字型: "5 个让我效率翻倍的 AI 工具✨"
  - 反差型: "月薪 3k 也能用的 AI 神器，免费！"
  - 痛点型: "别再花冤枉钱了！这些 AI 工具全免费"
  - 第一人称: "我用 AI 做副业，第一个月赚了 3000"
- **正文结构**:
  ```
  钩子(1-2句，点出痛点/价值)
  → 分点清单 (1. 2. 3. 每点配emoji)
  → 总结/收藏引导
  → tag 标签
  ```

## Formatting conventions
- **段间空行** — 大量留白，密集文字没人看
- **emoji 做小标题** — "✨ 1. xxx" "💡 2. xxx"
- **数字加粗或用【】** — 【免费】【亲测】【2026最新】
- **配图 6-9张** — 小红书是图文平台，图片比文字重要。第一张是封面（决定点击率）。**写完正文要单独生成 `xiaohongshu-images.md`（见下方「配图要求」）**
- **图片风格** — 干净、有文字标注、手机截图、对比图
- **文末标签** — 6-10个 #tag，mix 大词(#AI工具) + 长尾(#效率神器推荐)

## 配图要求（生成 `xiaohongshu-images.md`）

小红书是**图驱动**——图就是帖子本身，**第 1 张决定点击率**。`platform-writing` 写完正文后，**单独生成一份 `xiaohongshu-images.md`**（与通用 `images.md` 分开），含 6–9 张小红书专属配图 prompt。

- **比例**：**3:4 竖版（1080×1440）** 为主；最后一张引导图可 1:1。**不要**用公众号的 16:9。
- **首图 = 封面 = 点击率**：用最有冲击力的内容——反常识金句 / 最反差的对比 / 最值得收藏的清单。建议给 **2 个首图候选**，让用户 A/B 测。
- **文字卡为主**：每张一个核心信息，大标题 + 少量副文；标题至少占画面 1/4（手机信息流里图很小，字要大）。
- **全套风格统一**：同一套调色板 / 质感，别一张亮一张暗——视觉一致才显专业。
- **中文一律后期叠**：AI 直出中文基本乱码——prompt 生成**无字底图**，再用 Figma / Canva 叠中文。每张图在 prompt 旁列出 **"叠字内容"**（要贴上去的文案）。
- **每张配齐**：用途、比例、叠字内容、画面描述、可直接喂出图工具的 prompt。

**典型 8 张结构**（按内容裁剪到 6–9）：封面金句 / 一句话定义 / 核心反差对比 / 干货框架图 / 清单（最易收藏）/ 情绪警示 / 结论金句 / 关注引导。

> **清单型图**（"N 条标准""X 个工具"）**收藏率最高**——至少做一张，且可作首图候选。

## Hook patterns
- "姐妹们！我发现了一个神器..."
- "谁懂啊！终于不用花冤枉钱了"
- "用了一周，我后悔没早点知道"
- "免费！免费！免费！重要的事说三遍"

## CTA patterns
- "觉得有用记得❤️+⭐，下次找不到了"
- "评论区告诉我你想看什么"
- "关注我，每天分享一个 AI 小技巧✨"

## Tag strategy
- 大流量词: #AI #效率 #干货分享
- 中流量词: #AI工具 #宝藏app #副业
- 长尾词: #免费ai工具推荐 #学生党必备

## Common mistakes
- ❌ 太正式 — 小红书不需要书面语，要像朋友圈
- ❌ 文字太长 — 超过 800字完读率暴跌
- ❌ 图片太少 — 少于 3张会被算法降权
- ❌ 没有 tag — 流量减半
- ❌ emoji 太少或太假 — 小红书没有 emoji 显得格格不入
- ❌ 像广告 — 硬广会被限流，要软种草
