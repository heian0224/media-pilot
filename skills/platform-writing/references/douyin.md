# 抖音 (Douyin) Short-Video Script Guide

Short, hook-driven spoken scripts. The viewer's thumb is on the swipe — you have **3 seconds** to earn the next 30. Every word must earn its place.

## Voice & Tone
- **口语化** — 像跟朋友说话，不是念稿
- **有力/直接** — 短句，节奏快，不废话
- **情绪化** — 有情绪钩子（惊讶/愤怒/共鸣/好奇）
- **少书面词** — "因此/综上所述"全部去掉，换成"所以/说白了"

## Length & Structure
- **口播 150-400字** — 对应 30-90秒视频，超过 90秒完播率下降
- **结构 (黄金公式)**:
  ```
  钩子(0-3秒): 一句话让观众停下滑动
  → 痛点/冲突(3-10秒): 为什么这事跟我有关
  → 主体(10-60秒): 核心内容，分点说
  → 高潮/反转(可选): 强化记忆点
  → CTA(最后3-5秒): 关注/评论/点赞
  ```

## Hook patterns (前3秒最重要)
- **反常识**: "别再学 Python 了！2026 最该学的是这个"
- **数字冲击**: "99% 的人都不知道，AI 已经能..."
- **提问**: "月薪 5000 和月薪 5 万的程序员，差在哪？"
- **故事**: "昨天我同事被裁了，原因让我震惊"
- **冲突**: "用 AI 写代码被老板骂了？可能是你方法错了"
- **制造悬念**: "看到最后，你会感谢我"

## Script example
```
【钩子 3s】用了 3 年 ChatGPT，我发现 99% 的人都用错了！

【痛点 5s】你是不是也这样——问个问题，AI 给的答案又长又没用？

【主体 30s】其实，真正的高手都用这 3 个技巧：
第一，给角色。不要说"帮我写文案"，要说"你是一个有 10 年经验的小红书运营..."
第二，给例子。先喂一个你满意的样本，AI 立刻就懂你要什么。
第三，给约束。字数、风格、语气，全部限定死。

【高潮 10s】我用这 3 招，写文案从 1 小时缩短到 5 分钟。

【CTA 3s】关注我，每天一个 AI 实用技巧！
```

## Formatting conventions
- **写成口播脚本** — 不是文章，是"说的话"
- **标注时长** — 【钩子 3s】【主体 30s】方便拍摄时控时
- **短句为主** — 一句话别超过 15字，方便一口气说完
- **标点提示节奏** — 感叹号、省略号、问号都用上，提示语气起伏
- **配字幕建议** — 抖音视频都带字幕，关键字可以加粗/变色

## CTA patterns (抖音特有)
- "评论区扣 1，我私信你教程"
- "点赞过 1000，下期讲怎么变现"
- "关注我，主页有合集"
- "收藏起来，下次找不到"

## Hashtag 策略
- 视频文案里带 3-5个 #tag
- 大词 #AI + 领域词 #AI编程 + 时效词 #2026
- 参与挑战话题（如果有合适的）

## Common mistakes
- ❌ 开头太慢 — 前 3秒没钩子，80% 划走
- ❌ 像念稿 — 没有情绪起伏，观众睡着
- ❌ 太长太满 — 想说的太多，反而啥都没说清
- ❌ 没有 CTA — 观众看完了就划走，不关注
- ❌ 书面语 — "综上所述"这种词在抖音是灾难
- ❌ 信息密度太低 — 30秒只说了一个点，浪费时长

## 抖音 vs B站脚本区别
- 抖音更短、更快、更情绪化（30-90秒）
- B站更长、更知识、更详细（5-15分钟）
- 同一个选题，抖音版是"精华钩子版"，B站版是"完整深度版"
